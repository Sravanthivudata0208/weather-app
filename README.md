# Weather App

This is a simple Weather Application built using **HTML, CSS, and JavaScript**.
It allows users to search for a city and view the current weather information.

## Features

* Search weather by city name
* Displays temperature
* Displays weather condition
* Simple and clean UI

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Weather API

## Project Structure

weather-app
│── index.html
│── style.css
│── script.js

## How to Run

1. Clone or download the repository
2. Open **index.html** in your browser
